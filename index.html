Tap Ranking Monitor

TikTok Liveの「いいね（タップ）」ランキングをリアルタイムで可視化。
OBSに組み込んで、配信のエンゲージメントを最大化しよう。

GitHubでダウンロード

Features

⚡ リアルタイム更新

視聴者のタップを瞬時に検知し、ランキングを自動で並び替えます。熱心なファンを逃さずピックアップ。

🎨 OBS完全対応

ブラウザソースとしてOBSに追加するだけで、透過背景の美しいUIを配信画面にオーバーレイできます。

🛠 簡単セットアップ

複雑な設定は不要。Node.jsでサクッと立ち上げて、あなたのTikTok IDを入力するだけですぐに使えます。

Quick Start

1
リポジトリをクローン
git clone https://github.com/cooowabungaaa-hue/tiktok-tap-counter-v2.git
2
依存関係をインストール
cd tiktok-tap-counter-v2
npm install
3
サーバーを起動してOBSに追加
node server.js

表示されたローカルホストのURLを、OBSの「ブラウザソース」に追加してください。
